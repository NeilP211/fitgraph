# FitGraph

Most fashion recommenders surface *similar* items — "you liked this shirt, here's another shirt." FitGraph takes a different angle: it learns *compatibility* from the way garments are actually co-worn in real outfits, building a graph where nodes are clothing items and edges encode polyvore co-occurrence. A GNN then scores whether an arbitrary combination of pieces works together as an outfit.
